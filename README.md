![1](https://github.com/user-attachments/assets/644f09a1-b4b3-4d8c-a938-9a541efd77e7)
### 2. Click to Raw button
![2](https://github.com/user-attachments/assets/4f024220-7ffd-49dc-a23b-a63702271e98)

![3](https://github.com/user-attachments/assets/71073a0d-6294-4427-b995-f853c447636c)

![4](https://github.com/user-attachments/assets/3dc1c2f9-a853-4a88-af3f-0131db813a2d)
### 5. Click Add blocklist button
![5](https://github.com/user-attachments/assets/f9ab57fa-cb11-4b4e-8940-91ef703b1f28)
### 6. Click Add a custom list button
![6](https://github.com/user-attachments/assets/625cac52-3678-46c7-a8e5-d9211da57461)
### 7. Enter name of New blocklist and paste the URL on step 3 --> Save --> Done.
